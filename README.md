# Changelog Generator

Un générateur de changelog au format JSON/Markdown pour les projets github  
Le but principal du projet est de me faire tester et apprendre VueJs 2.0  

By Florian B. (Alias fkeloks)

## Utilisation :

Avec npm:
```
npm run dev // Mode dev
npm run build // Mode production
```
